Projeto 1: Calculadora de Idade em C++

Neste projeto, você aprenderá a criar um programa simples em C++ utilizando a função `printf` do Code::Blocks para calcular a idade de uma pessoa com base no ano atual e no ano de nascimento. Este é um ótimo exercício para iniciantes em programação C++ e ajudará você a entender os conceitos básicos de entrada e saída, operações aritméticas e variáveis.
